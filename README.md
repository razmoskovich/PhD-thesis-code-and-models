# PhD Thesis – Code and Models

Repository containing analysis scripts, statistical workflows, figure-generation code, and metabolic network models associated with my PhD thesis in marine sponge ecology and physiology.

## Repository structure

### Chapter 1 – Method comparison
Methodological comparison analyses, datalogger processing, Agisoft workflows, and temperature experiments.

### Chapter 2 – Allometry
Allometric analyses and associated statistical workflows.

### Chapter 3 – Feeding & Respiration
Scripts and analyses related to sponge feeding, respiration, and physiological flux measurements.

### Chapter 4 – Metabolic network model
Species-specific LIM input files, metabolic network analyses, and model-construction workflows.
